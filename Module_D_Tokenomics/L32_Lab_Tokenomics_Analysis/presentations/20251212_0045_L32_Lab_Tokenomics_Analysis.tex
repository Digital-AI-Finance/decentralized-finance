    \item Upcoming unlock events?
\end{itemize}

\begin{frame}{Step 4: Calculate Metrics}
\textbf{Fully Diluted Valuation (FDV):}
\[
\text{FDV} = \text{Current Price} \times \text{Max Total Supply}
\]

\textbf{Dilution Risk Ratio:}
\[
\text{Dilution Risk} = \frac{\text{FDV}}{\text{Market Cap}}
\]
\begin{itemize}
    \item Ratio > 2: High dilution risk
    \item Ratio 1-2: Moderate dilution
    \item Ratio $\approx$ 1: Low dilution
\end{itemize}

\textbf{NVT Ratio:}
\[
\text{NVT} = \frac{\text{Market Cap}}{\text{Daily Tx Volume (USD)}}
\]
\end{frame}

\begin{frame}{Step 5: Apply Howey Test}
\textbf{Four Prongs:}
\begin{enumerate}
    \item \textbf{Investment of Money?} Did investors pay?
    \item \textbf{Common Enterprise?} Pooled returns?
    \item \textbf{Expectation of Profits?} Marketed as investment?
    \item \textbf{Efforts of Others?} Team drives value?
\end{enumerate}

\vspace{0.3cm}
\textbf{Classification:}
\begin{itemize}
    \item All four = Likely Security
    \item Fails ``efforts of others'' = Likely Utility Token
\end{itemize}
\end{frame}

\begin{frame}{Step 6: Develop Investment Thesis}
\textbf{Structure Your Analysis:}
\begin{enumerate}
    \item \textbf{Summary:} 2-3 sentences on token purpose
    \item \textbf{Strengths:} Bullish tokenomics factors
    \item \textbf{Weaknesses:} Red flags and risks
    \item \textbf{Verdict:} Buy / Hold / Avoid with justification
\end{enumerate}

\vspace{0.3cm}
\textbf{Note:} Educational analysis, not financial advice
\end{frame}

\begin{frame}{Key Takeaways}
\begin{itemize}
    \item Tokenomics analysis combines quantitative metrics and qualitative assessment
    \item Distribution and vesting reveal insider alignment and dilution risk
    \item FDV/MC ratio is critical for understanding future dilution
    \item Howey Test helps classify utility vs. security tokens
    \item Always compare to similar projects for context
\end{itemize}
