## Grade Appeals

If you wish to appeal a grade:

1. **Wait 24 hours** before submitting appeal (reflection period)
2. **Submit written appeal** via email within 7 days of grade posting
3. **Include specific justification** with reference to rubric criteria
4. **Instructor reviews** and responds within 5 business days
5. **Decision is final** after instructor review

Appeals based solely on "I need a better grade" will not be considered. Appeals must reference specific rubric criteria and provide evidence.

## Academic Integrity Policy

All work submitted must be your own (or your team's for group projects). Violations include:

- Copying code from other students or teams
- Using AI-generated code without understanding and disclosure
- Plagiarizing documentation or whitepapers
- Sharing quiz answers or solutions
- Submitting work done by others

**First Violation**: Zero on assignment + academic integrity office notification
**Second Violation**: F in course + academic integrity office escalation

**Permitted**:
- Using OpenZeppelin libraries and standard implementations (with attribution)
- Consulting official documentation and tutorials (with citation)
- Using AI coding assistants IF you understand all code and disclose usage
- Discussing concepts with other students (but not sharing code)
